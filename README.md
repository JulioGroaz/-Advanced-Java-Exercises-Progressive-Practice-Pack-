# Advanced Java Exercises - Progressive Practice Pack

English
Overview
This folder contains a progressive set of advanced Java exercises, from easier
to intermediate-difficult. Each exercise has its own folder with a bilingual
README and a Main.java template to complete.

Exercises
01-streams-analytics - Stream aggregations and collectors on transactions.
02-immutability-and-records - Immutable pricing rules with immutable classes.
03-generics-and-variance - Generics variance with PECS, transfer, and max.
04-concurrency-executors - Executors, timeouts, and task results.
05-nio-file-indexer - NIO file tree traversal and word counting.

Requirements
Java 8 JDK (javac is required to compile).

How to run
Each exercise is standalone (no package). Compile and run from the exercise folder:

javac Main.java
java Main

Open the README.md inside each folder for the input/output details.
When you start, you will be asked to choose a language (IT/EN). Only the first 2 characters are considered, case-insensitive.

VS Code setup (avoid red errors)
Open one exercise folder at a time: File -> Open Folder... -> select 01-streams-analytics (or another).
If you keep src open, set java.project.sourcePaths to a single folder in .vscode/settings.json.
Update the folder name when you switch exercises.

Italiano
Panoramica
Questa cartella contiene un percorso di esercizi Java avanzati, dal piu facile
ad un livello intermedio difficile. Ogni esercizio ha la sua cartella con un
README bilingue e un template Main.java da completare.

Esercizi
01-streams-analytics - Aggregazioni con Stream e collectors su transazioni.
02-immutability-and-records - Regole di pricing immutabili con classi immutabili.
03-generics-and-variance - Varianza generics con PECS, transfer e max.
04-concurrency-executors - Executors, timeout e risultati dei task.
05-nio-file-indexer - Traversal NIO e conteggio parole.

Requisiti
JDK Java 8 (serve javac per compilare).

Come usare
Ogni esercizio e standalone (senza package). Compila ed esegui dalla cartella
dell esercizio:

javac Main.java
java Main

Apri il README.md dentro ogni cartella per i dettagli su input/output.
All avvio ti viene chiesto di scegliere una lingua (IT/EN). Si considerano solo i primi 2 caratteri, senza distinzione maiuscole/minuscole.

Setup VS Code (evita errori rossi)
Apri una sola cartella di esercizio alla volta: File -> Open Folder... -> seleziona 01-streams-analytics (o un altra).
Se tieni src aperta, imposta java.project.sourcePaths su una sola cartella in .vscode/settings.json.
Aggiorna il nome della cartella quando cambi esercizio.
